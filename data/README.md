# Dataset

## Download Instructions

1. Go to: https://www.kaggle.com/datasets/yasserh/loan-default-dataset
2. Click "Download" (you may need a Kaggle account)
3. Extract the downloaded ZIP file
4. Place `Loan_Default.csv` in this folder

## Expected File

After downloading, this folder should contain:

```
data/
├── README.md           (this file)
└── Loan_Default.csv    (download from Kaggle)
```

## Dataset Information

- **Source:** Kaggle
- **URL:** https://www.kaggle.com/datasets/yasserh/loan-default-dataset
- **Size:** ~250,000 records
- **Format:** CSV

## Key Features

The dataset contains borrower and loan information including:
- Loan ID and status (default/no default)
- Borrower demographics (age, income, employment)
- Loan details (amount, term, interest rate)
- Property and credit information
- Payment history indicators

See the Data Dictionary in the documentation folder for detailed column descriptions.
